افزودن kcfinder به ckeditor

برای افزودن kcfinder به ckeditor در فریمورک لاراول مراحل زیر رو انجام میدیم:

1- ابتدا kcfinder رو دانلود میکنیم.

2- فایل فشرده را در پوشه ریشه از پروژمون اکسترکت میکنیم( میتونیم در پوشه ریشه پوشه دیگه ای به نام editor بسازیم و پوشه های ckeditor و kcfinder رو در اون پوشه قرار بدیم )

3- کدهای زیر رو در تابع config از فایل config.js قرار میدیم:

CKEDITOR.editorConfig = function( config ) {

    $.ajaxSetup({
        headers: {
            'X-CSRF-TOKEN': $('meta[name="csrf-token"]').attr('content')
        }
    });

    $.ajax({
        type: 'POST',
        url: '/geturl',
        dataType: 'json',
        success: function (data) {
            var path = data.url;
            config.filebrowserBrowseUrl = path + '/browse.php?opener=ckeditor&type=files';
            config.filebrowserImageBrowseUrl = path + '/browse.php?opener=ckeditor&type=images';
            config.filebrowserFlashBrowseUrl = path + '/browse.php?opener=ckeditor&type=flash';
            config.filebrowserUploadUrl = path + '/upload.php?opener=ckeditor&type=files';
            config.filebrowserImageUploadUrl = path + '/upload.php?opener=ckeditor&type=images';
            config.filebrowserFlashUploadUrl = path + '/upload.php?opener=ckeditor&type=flash';
        },
        error: function (data) {
            var a = '<pre class="alert alert-danger">';
            $.each(data, function(key, val){
                a += val + '<br />';
            });
            $('.alert').append(a + '</pre>');
        }
    });
}

 

4- کدهای زیر را در فایل kcfinder/conf/config.php قرار میدیم:

<?php

$url = '../../../../../../uploads';

$uploadPath = dirname(dirname(dirname(dirname(dirname(dirname(__FILE__)))))) . '/uploads';

$dir = ($_SERVER['SERVER_NAME'] == 'localhost' ? $uploadPath : '');

// برای اینکه سشن کش نشود نامش را از طریق تایم تصادفی قرار میدهیم
$sessionVar = 'Kcfinder_' . time();

$_CONFIG = array(


    'disabled' => false,
    'uploadURL' => $url,
    'uploadDir' => $dir,
    'theme' => "default",

    'types' => array(
		// The folowing directory types are just for an example
		'files'     => "",

		'images'    => "*img",
		'mimages'   => "*mime image/gif image/png image/jpeg",
		'notimages' => "*mime ! image/gif image/png image/jpeg"
	),

// IMAGE SETTINGS

    'imageDriversPriority' => "imagick gmagick gd",
    'jpegQuality' => 100,
    'thumbsDir' => ".thumbs",

    'maxImageWidth' => 800,
    'maxImageHeight' => 800,

    'thumbWidth' => 100,
    'thumbHeight' => 100,

    'watermark' => "",


// DISABLE / ENABLE SETTINGS

    'denyZipDownload' => true,
    'denyUpdateCheck' => false,
    'denyExtensionRename' => false,


// PERMISSION SETTINGS

    'dirPerms' => 0777,
    'filePerms' => 0644,

        'files' => array(
            'upload' => true,

            'create' => true,

   'disabled' => true,
);

?>
