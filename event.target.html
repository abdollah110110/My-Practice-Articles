event.target

event.target همان تگی که رویش کلیک شده است را برمیگرداند.

در مثال زیر به کلاس و parent های تگ کلیک شده دسترسی پیدا کردیم:

                $('#sidebar').on('hide.bs.collapse', function() {
                    console.log(event.target);
                    var clicked = $(event.target).hasClass('sidebar-toggler');
                    var clickedParents = $(event.target).parents().hasClass('sidebar-toggler');
                    if (clicked === true || clickedParents === true) {
                        $('#content').removeClass('col-lg-9');
                        $('#sidebar-toggler').removeClass('bg-info text-white');
                    }
                });
