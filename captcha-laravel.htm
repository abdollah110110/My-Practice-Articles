نصب Captcha در لاراول

برای استفاده از تصویر امنیتی در فرم ها می توانیم از هر captcha ایی که با لاراول integrate شده باشد استفاده کنیم به طور مثال یکی از بهترین captcha ها reCaptcha گوگل است که پکیج های مربوط به لاراول آن را می توانید با جستجو پیدا کنید.

ما در این مثال از captcha mews استفاده می کنیم.

برای نصب captcha اول باید پیش نیازهای composer و laravel رو نصب کنیم.

1- باید fxp/composer-asset-plugin نصب بشه با دستور زیر: 

composer global require "fxp/composer-asset-plugin:^1.2.0"

 

2- باید دستور زیر رو اجرا کنیم:

composer update

 

3- باید Git رو از این ادرس دانلود و نصب کنیم.

 

4- باید intervention image رو نصب کنیم با دستور زیر: 

composer require intervention/image:2.3

 

5- دستور زیر را اجرا کنیم تا فایل image.php در پوشه config ایجاد شود: 

php artisan vendor:publish

 

6- در فایل image.php مقدار driver رو تعیین میکنیم: 

return array(
 
   // gd or imagick
   'driver' => 'gd'
 
);

 

مثال:

<?php
$img = \Image::canvas(200, 200, '#000');
 
// draw a red line with 5 pixel width
$img->line(10, 10, 190, 190, function ($draw) {
   $draw->color('#0f0');
});
 
// draw a filled blue circle with red border
$img->circle(90, 145, 145, function ($draw) {
   $draw->background('#00f');
   $draw->border(3, '#f00');
});
 
// use callback to define details
$img->text('Intervention Image', 95, 95, function($font) {
   $font->file(base_path() . '/public/fonts/font.otf');
   $font->size(16);
   $font->color('#fff');
   $font->align('center');
   $font->valign('top');
   $font->angle(45);
});
 
// draw transparent rectangle with 2px border
$img->rectangle(15, 15, 90, 90, function ($draw) {
   $draw->background('rgba(150, 0, 100, 0.5)');
   $draw->border(2, '#f00');
});
 
$img->encode('jpg');
$base64 = 'data:image/jpg;base64,' . base64_encode($img);
?>
<img src="{!! $base64 !!}" />

 

مستندات Intervention Image  رو مطالعه نمایید.

 

یه چیزی اینجا داخل پرانتز بگم: ممکنه وقتی مسیر کلاس جدیدی در aliases در فایل config/app.php اضافه میکنیم، اون کلاس در پروژه شناسایی نشه و خطای Not found بگیریم، برای حل این مشکل اول دستور زیر را اجرا میکنیم:

php artisan config:clear

و بعد این دستور: 

php artisan config:cache

 

حالا که پیش نیازها رو نصب کردیم، نصب captcha شروع میکنیم و برای این کار به دایرکتوری پروژه خود میرویم و دستور زیر رو اجرا میکنیم:

composer require mews/captcha

 

بعد از اینکه نصب شد به پوشه config رفته و فایل app.php را باز میکنیم و به انتهای آرایه providers خط زیر را اضافه کنید: 

Mews\Captcha\CaptchaServiceProvider::class,

 

و به انتهای آرایه aliases هم خط زیر را اضافه میکنیم: 

'Captcha' => Mews\Captcha\Facades\Captcha::class,

 

در انتها دستور زیر را در cmd تایپ و اجرا میکنیم: 

php artisan vendor:publish --provider="Mews\Captcha\CaptchaServiceProvider"

 

نکته: باید GD2 PHP Extension داخل سرور شما فعال باشد وگرنه کپچا کار نخواهد کرد بنابراین در صورتی که آن فعال نیست در فایل php.ini علامت ; را از ابتدای خط php_gd2.dll بردارید و یک بار apache خود را ریستارت کنید.

 

حالا اگر فایل captcha.php در پوشه config را باز کنید می توانید تنظیمات مربوط به تصویر captcha را تغییر دهید. به طور مثال با length تعداد حروف و اعدادی که نمایش می دهد را مشخص می کنیم.

 

حالا به فرم تماس با ما، captcha اضافه میکنیم. در فایل ویو contact.blade.php کدهای زیر رو اضافه میکنیم:

<div class="form-group <?= (isset($errors) && $errors->has('captcha') ? 'has-error' : '') ?>">
    <div class="row">
        <div class="col-xs-12 col-sm-4">
            {!! Form::label('captcha', 'کد امنیتی', ['class' =>'control-label']) !!}
            {!! Form::text('captcha', null, ['class' => 'form-control', 'placeholder' => 'کد امنیتی']) !!}
            @if(isset($errors) && $errors->has('captcha'))
                {!! $errors->first('captcha', '<span class="error">:message</span>') !!}
            @endif
        </div>
        <div class="col-xs-1 col-sm-1">
            <button type="button" id="refresh" class="btn btn-sm btn-info" title="تغییر کد امنیتی">
                <i class="glyphicon glyphicon-refresh"></i>
            </button>
        </div>
        <div class="col-xs-11 col-sm-7" id="captcha-div">
            {!! Captcha::img('flat') !!}
        </div>
    </div>
</div>

همانطور که مشاهده می کنید با استفاده از تابع captcha_img تصویر امنیتی را ایجاد می کنیم. این captcha در حالت های flat و inverse و mini دارای ابعاد و پیکربندی های متفاوتی هست که به عنوان پارامتر باید به متد img ان را پاس بدهیم. اگر هم آن را خالی بگذارید پیکربندی default اعمال می شود.

یک textbox به نام captcha هم ایجاد کردیم تا کاربر کد تصویر امنیتی را در آنجا تایپ کند.

همچنین یک کلید button برای refresh کردن تصویر امنیتی هم در کنار captcha قرار دادم که هر بار با کلیک کردن روی آن تصویر reload شود، البته باید کد jquery که برای reload شدن تصویر امنیتی نوشتم را به انتهای قالب master.blade.php اضافه کنیم: 

$('#refresh').on('click', function () {
    var time = new Date().getTime();
    var oldCaptchaPath = $('img[alt=captcha]').attr('src');
    // مسیر سورس کپچا را از جایی که علامت سوال دارد
    // به دو قسمت میشکند و قسمت اول را برمیدارد
    // و یک علامت سوال و زمان جاری را به آن می افزاید تا کش نشود
    var newCaptchaPath = oldCaptchaPath.split('?', 1) + '?' + time;
    $('img[alt=captcha]').attr('src', newCaptchaPath);
});

 

بعد rule زیر را هم به rule های اعتبارسنجی فرم contact اضافه میکنیم:

'captcha' => 'required|captcha',

 

اگر کدی که کاربر وارد کند با کد تصویر امنیتی یکسان نباشد پیغام خطایی نیز باید چاپ شود بنابراین پیغام خطا را در فایل validation.php در پوشه resources/lang/fa اضافه میکنیم: 

'captcha' => 'کد امنیتی را صحیح وارد کنید',
