مجوز دسترسی با استفاده از Gate در laravel

از Gateها برای بررسی مجوز داشتن کاربران برای دسترسی به بخشی از سایت استفاده میکنیم. اگر یک لینک بصورت زیر داشته باشیم:

<a href="/posts/{{ $post->id }}">{{ $post->title }}</a>

روت مربوط به این لینک را بصورت زیر تعریف میکنیم:

Route::get('/posts/{post}', 'PostController@show')->name('show.post');

مراحل تعریف Gate برای روت بالا بصورت زیر خواهد بود:

1- در مسیر app\Providers، و در فایل AuthServiceProvider.php کدهای زیر را وارد میکنیم :

public function boot()
{
    $this->registerPolicies();

    Gate::define('show-post', function($user, $post){
        return $user->id == $post->user_id;
    });
}

در این کد اطلاعات کاربر توسط خود gate به تابع ارسال میشود اما اطلاعات پست را ما باید ارسال کنیم.

2- متد show را در کنترلر PostController بصورت زیر تعریف میکنیم:

public function show(Post $post) {

    auth()->loginUsingId(1);

    if(Gate::allows('show-post', $post))
        return view('posts.show', compact('post'));

    abort(403, 'You do not have permission to access.');
}

3- فایل resources\views\errors\403.blade.php را ایجاد میکنیم:

@extends('layouts.welcome')

@section('content')
<div class="alert alert-danger text-center">
    <h3>{{ $exception->getMessage() }}</h3>
</div>
@endsection

اگر بخواهیم داشتن مجوز دسترسی را در ویوها بررسی کنیم به شکل زیر از can و cannot استفاده میکنیم:

@foreach($posts as $post)
    <div class="alert alert-success" style="direction:ltr;">
        
@can('show-post', $post)
       <h2><a href="/posts/{{ $post->id }}">{{ $post->title }}</a></h2>
@endcan
        
@cannot('show-post', $post)
     <h2>{{ $post->title }}</h2>
@endcannot
        
<p>{{ $post->body }}</p>
    </div>
@endforeach
