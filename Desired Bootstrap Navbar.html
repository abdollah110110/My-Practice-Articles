2- اگر میخواهیم از حالت md به پایین navbar-toggler را مشاهده کنیم کلاس navbar-expand-lg را به navbar-expand-md تغییر میدهیم.

<nav class="navbar navbar-expand-md navbar-light bg-light">

3- اگر بخواهیم navbar بصورت دائمی در بالای صفحه قرار داشته باشد کلاس fixed-top را به آن اضافه میکنیم.

<nav class="navbar navbar-expand-lg fixed-top navbar-light bg-light">

4- برای اینکه لینکها و فرم جستجو در دو طرف قرار بگیرند کلاس justify-content-between را به کلاس collapse navbar-collapse اضافه میکنیم.

<div class="collapse navbar-collapse justify-content-between" id="navbarSupportedContent">

5- برای اینکه محدوده بین navbar-brand و فرم جستجو بصورت کامل با لینکها پر شود، کلاس flex-fill را به کلاس navbar-nav اختصاص میدهیم.

<ul class="navbar-nav flex-fill">

6- برای اینکه فاصله بین لینکها مساوی باشد کلاس flex-fill را به کلاسهای nav-item اختصاص میدهیم.

<li class="flex-fill nav-item">

9- کدهای jquery لازم برای navbar بصورت زیر خواهند بود.

$(document).ready(function() {

        // Show dropdown in slide mode when whidth < 768
        $('.dropdown').on('show.bs.dropdown', function() {
            if($(window).width() < 768){
                    $(this).find('.dropdown-menu').first().stop(true, true).slideDown();
                }    
        });
        $('.dropdown').on('hide.bs.dropdown', function() {
                if($(window).width() < 768){
                        $(this).find('.dropdown-menu').first().stop(true, true).slideUp();
                }    
        });

        // Change navbar styles
        $(window).scroll(function() {
                if($(window).scrollTop() > 60) {
                        $('.navbar').addClass('py-0');
                } else {
                        $('.navbar').removeClass('py-0');
                }
        });

});
