Exception

خطاهای بوجود آمده در متدهای کلاسها را مدیریت میکند.

مثال:


<?php 

class Math{

    public static function sqrtval($value){

        if($value < 0){

            throw new Exception("Can't (value < 0).", 400);

        }

        echo '<p>Value: '. sqrt($value) . '</p>' . PHP_EOL;

    }

} 



try{

    Math::sqrtVal(-36);

}

catch(Exception $e){

    echo '<p><b>' . $e->getCode() . ': </b>' . $e->getMessage() . '</p>' . PHP_EOL; 

}

finally{

    echo '<p>کدی که به هر حال باید اجرا شود</p>' . PHP_EOL;

}

?>

خروجی مثال بالا:

400: Can't (value < 0).

کدی که به هر حال باید اجرا شود
