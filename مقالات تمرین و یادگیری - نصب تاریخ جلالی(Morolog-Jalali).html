نصب تاریخ جلالی(Morolog-Jalali)

برای نصب و نمایش تاریخ جلالی به صفحه Morolog - Jalali رفته و طبق آموزش نصب را انجام میدهیم. بعد یک مایگریشن ایجاد میکنیم و با دستورات زیر دو فیلد creat_time و update_time را به جدول مورد نظر اضافه میکنیم:

php artisan make:migration add_jalali_fields_to_articles_table --table=articles
public function up()
{
    Schema::table('articles', function (Blueprint $table) {
        $table->string('create_time')->after('active')->default(0);
        $table->string('update_time')->after('active')->default(0);
    });
}

public function down()
{
    Schema::table('articles', function (Blueprint $table) {
        $table->dropColumn('create_time');
        $table->dropColumn('update_time');
    });
}

در کنترلر برای استفاده از این فیلدها دو کلاس زیر را use میکنیم:

use Carbon\Carbon;
use Morilog\Jalali\Jalalian;

بعد در متد store فیلد create_time را مقداردهی میکنیم:

$article->create_time = Jalalian::fromCarbon(Carbon::now());

در متد update هم فیلد update_time را مقداردهی میکنیم:

$article->update_time = Jalalian::fromCarbon(Carbon::now());

و در نهایت به روش زیر در ویو نمایش میدهیم:

<div class="clearfix d-flex justify-content-center mb-3">
    <div class="px-3">
        <span>تاریخ ایجاد: </span>
        <small class="text-muted">{{ $article->create_time }}</small>
    </div>
    <div class="px-3">
        <span>تاریخ ویرایش: </span>
        <small class="text-muted">{{ $article->update_time }}</small>
    </div>
</div>
