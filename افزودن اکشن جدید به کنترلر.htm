افزودن اکشن جدید به کنترلر

فرض کنید فرمی داریم که از طریق آن id های چند کامنت را انتخاب میکنیم، برای این کار فرمی را بصورت زیر تعریف میکنیم:

<form method="POST" action="{{ Url('comments/newaction') }}">
{{ csrf_field() }}
<input name="comments[]" value="2" type="checkbox">
<input name="comments[]" value="3" type="checkbox">
<input name="comments[]" value="5" type="checkbox">
<input name="comments[]" value="8" type="checkbox">
</form>

 

در فایل route.php دستورات زیر را مینویسیم:

Route::post('comments/newaction', 'CommentsController@newaction');
Route::resource('comments', 'CommentsController');

به عنوان یک مطلب داخل پرانتز اینو میگم: نقطه ای که اینجا وجود داره اینه که اتریبیوت method در فرم را هر چی انتخاب کنیم باید route رو هم با همون بنویسیم:

<form method="PUT" action="{{ Url('comments/newaction') }}">
// ...
</form>

// توی فایل route.php
Route::put('comments/newaction', 'CommentsController@newaction');

و همینطور route را بصورت controller/action تعریف میکنیم.

 

و در Controller اکشن زیر را تعریف میکنیم:

public function newaction(Request $request){
    foreach($request->comments as $id){
        $comment = Comment::find($id);
        $comment->delete();
    }
    return \Redirect::route('comments.index');
}

در واقع متد newaction کار همان destroy یا delete را انجام میدهد.
