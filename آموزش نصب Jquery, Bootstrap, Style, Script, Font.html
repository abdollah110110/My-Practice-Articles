دانلود و استفاده از Bootstrap

<link href="css/bootstrap.min.css" rel="stylesheet">

دانلود و نصب آیکن های font awsome

ابتدا font awsome را از این صفحه دانلود میکنیم. بعد از اکسترکت کردن فایل فشرده، سه فولدر css و js و webfonts  را در مسیر font\fontawsome از پروژه قرار میدهیم.

در فولدر css به غیر از all.min.css بقیه فایلها را حذف میکنیم.

در فولدر js به غیر از all.min.js بقیه فایلها را حذف میکنیم.

فایل all.min.css را در صفحه لود میکنیم:

<link href="font/fontawesome/css/all.min.css" rel="stylesheet">

فایل all.min.js را در صفحه لود میکنیم:

<script src="font/fontawesome/css/all.min.js"></script>

برای دیدن نحوه استفاده از آیکنها به این صفحه میرویم و به شکل زیر آیکنهای مورد نیاز را استفاده میکنیم:

<i class="fas fa-caret-down"></i>
