تنظیمات mysql برای پروژه لاراول

صفحه اصلی مقالات

 

برای اینکه دستورات مایگریشن دچار ارور نشوند یکی از دو روش زیر را انجام میدهیم:

 

  • در این روش بصورت پیشفرض پشتیبانی از ایموجیها را لاراول در نظر گرفته است و تنظیمات mysql بصورت زیر است:
'charset' => 'utf8mb4',
'collation' => 'utf8mb4_unicode_ci',

و کالیشن دیتابیسی که میسازیم را utf8mb4_unicode_ci قرار میدهیم و همچنین در فایل AppServiceProvider.php کد زیر را در متد boot قرار میدهیم:

use Illuminate\Support\Facades\Schema;

Schema::defaultStringLength(191);

 

  • در این روش تنظیمات mysql را به شکل زیر تغییر میدهیم:
'charset' => 'utf8',
'collation' => 'utf8_persian_ci',

و کالیشن دیتابیسی که ساختیم را utf8_persian_ci قرار میدهیم.
