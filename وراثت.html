وراثت

ارث بری فیلدها و متدها:
فیلدها و متدهای private به ارث برده نمیشوند، از بیرون از کلاس قابل دسترسی نسیتند
فیلدها و متدهای public و protected به ارث برده میشوند.
فیلدها و متدهای protected به ارث برده میشوند اما از بیرون از کلاس قابل دسترسی نیستند، برای حفظ امنیت.
فیلدها و متدهای public هم به ارث برده میشوند و هم از بیرون کلاس قابل دسترسی هستند.

extends:
با استفاده از این کلمه یک کلاس از کلاس دیگر به ارث برده میشود و در واقع فرزند ان کلاس میشود.

parent:
با استفاده از این کلمه متدی که در کلاس والد هست را در متدی با همان نام در کلاس فرزند فراخوانی و اجرا میکنیم.
تغییر دادن رفتار متدهای کلاس والد توسط متدهایی در کلاس فرزند را چندریختی کردن میگویند.

final:
اگر در ابتدای متد نوشته شود یعنی نمیتوان آن متد را در کلاس فرزند تغییر داد و در واقع قابل چندریخته شدن نیست.

instanceof:
چک میکند که آیا شیء از نوع کلاس گفته شده است یا نه؟

get_class:
نام کلاسی که شیء از آن ساخته شده است را برمیگرداند. اگر get_class در کلاس والد باشد و ما از طریق شیء فرزند متد مربوط به آن را صدا بزنیم باید پارامتر this$ را به آن ارسال کنیم تا نام کلاس فرزند برگردانده شود. 
public function getClassName(){
	return get_class($this);
}

مثال:


<?php 

class Father

{

    protected function getRandomNumber()

    {

        return rand(1, 100);

    }

    

    public function randomNumber()

    {

        echo '<p style="color:red;">Father random number: ' . $this->getRandomNumber() . '</p>' . PHP_EOL;

    }

    

    final public function rangeNumber(){

        $numbers = implode(' - ', range(1, 10));

        echo '<p>Father range numbers: ' . $numbers . '</p>' . PHP_EOL;

    }

    

    public function getClassName(){

        echo '<p>Class name: ' . get_class($this) . '</p>' . PHP_EOL;

    }

}



class Child extends Father

{

    public function randomNumber()

    {

        parent::randomNumber();

        echo '<hr /><p style="color:green;">Child random number: ' . $this->getRandomNumber() . '</p>' . PHP_EOL;

    }

}



$numbers = new Child;

$numbers->randomNumber();

if($numbers instanceof Child){

    $numbers->rangeNumber();

}

$numbers->getClassName();

?>
