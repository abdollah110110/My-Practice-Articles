چک کردن admin یا user بودن کاربر لاگین شده

برای اینکه بعد از لاگین کردن کاربر تشخیص دهیم کاربر وارد شده Admin است یا user معمولی، یک Middleware ایجاد میکنیم.

1- ایجاد middleware با استفاده از دستور زیر در خط فرمان:

php artisan make:middleware CheckAdminOrUser

2- به فایل app\Http\Middleware\CheckAdminOrUser.php میرویم و به شکل زیر تغییرش میدهیم:

<?php

namespace App\Http\Middleware;

use Closure;

class CheckAdminOrUser
{
    public function handle($request, Closure $next)
    {
        if(auth()->check() && auth()->user()->isAdmin()){
            return $next($request);
        }
        return redirect('/');
    }
}

3- در فایل app\Http\Kernel.php مسیر middleware را وارد میکنیم:

protected $routeMiddleware = [
    // ...
    'checkAdmin' => \App\Http\Middleware\CheckAdminOrUser::class,
];

4- در مدل User متد زیر را وارد میکنیم:

public function isAdmin()
{
    return $this->level == 'admin' ? true : false;
}

5- با قرار دادن روتهایی که باید برای admin نمایش داده شوند در یک Route::group ، مانع از ورود سایر کاربران به بخشهای مدیریتی سایت میشویم:

<?php

Route::group(
    [
//        'namespace' => '',
//        'prefix' => '',
        'middleware' => ['checkAdmin', 'auth'],
    ],
    function(){
        //Categories Routes
        Route::get('/categories', 'CategoryController@index')->name('categories');
        Route::get('/categories/create', 'CategoryController@create')->name('categories.create');
        Route::post('/categories/store', 'CategoryController@store')->name('categories.store');
        Route::get('/categories/{category}/show', 'CategoryController@show')->name('categories.show');
        Route::delete('/categories/destroy/{category}', 'CategoryController@destroy')->name('categories.destroy');
        Route::get('/categories/{category}/edit', 'CategoryController@edit')->name('categories.edit');
        Route::put('/categories/{category}/update', 'CategoryController@update')->name('categories.update');

        //Articles Routes
        Route::get('/articles', 'ArticleController@index')->name('articles');
        Route::get('/articles/create', 'ArticleController@create')->name('articles.create');
        Route::post('/articles/store', 'ArticleController@store')->name('articles.store');
        Route::delete('/articles/destroy/{article}', 'ArticleController@destroy')->name('articles.destroy');
        Route::get('/articles/{article}/edit', 'ArticleController@edit')->name('articles.edit');
        Route::put('/articles/{article}/update', 'ArticleController@update')->name('articles.update');
    }
);

// Category selected route
Route::get('/articles/category/{category}', 'ArticleController@selected')->name('category.selected');

// Articles show Route
Route::get('/articles/{article}/show', 'ArticleController@show')->name('articles.show');

// Home routes
Route::get('/slot-view', function(){
    return view('site.slot-view');
});
Route::get('/', 'HomeController@index')->name('home');
Route::get('/home/activation/{token}', 'HomeController@activation')->name('activation');


Route::group(
    [
        'namespace' => 'Auth', 
        'prefix' => 'auth'
    ],
    function(){
        // Registration Routes...
        Route::get('/register', 'RegisterController@showRegistrationForm')->name('register');
        Route::post('/register', 'RegisterController@register')->name('register.post');


        // Login-Logout Routes...
        Route::get('/login', 'LoginController@showLoginForm')->name('login');
        Route::post('/login', 'LoginController@login')->name('login.post');
        Route::post('/logout', 'LoginController@logout')->name('logout');


        // Password Reset Routes...
        Route::get('/password/reset', 'ForgotPasswordController@showLinkRequestForm')->name('password.request');
        Route::post('/password/email', 'ForgotPasswordController@sendResetLinkEmail')->name('password.email');
        Route::get('/password/reset/{token}', 'ResetPasswordController@showResetForm')->name('password.reset');
        Route::post('/password/reset', 'ResetPasswordController@reset')->name('password.update');
    }
);

6- در navbar یا هر جای دیگر سایت لینکها یا کدهایی که فقط مخصوص مدیر است را بین شرط زیر قرار میدهیم:

@if(Auth::user()->isAdmin())
<h3 class="mb-5">سلام <b class="text-success">{{ Auth::user()->name }}</b> گرامی</h3>
<p class="mt-5">برای مدیریت سایت میتوانید از لینک های زیر استفاده نمایید:</p>
<a class="btn btn-info mt-3" href="/categories">ورود به بخش موضوعات</a>
<a class="btn btn-primary mt-3" href="/articles">ورود به بخش مقالات</a>
@else
<p class="mt-5">به وب سایت من خوش آمدید</p>
@endif
