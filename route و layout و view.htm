کار با route و layout و view

در مواقعی که صفحه مورد نظر از کنترلر استفاده نمیکند، از route هایی که شامل یک تابع بدون نام هستند استفاده میکنیم:

Route::get('/', function () {
    return view('home');
});

این route یک view به اسم home.blade.php را وقتی که url در صفحه اول سایت هست، نشان میدهد.

 

layout ها و view ها را در مسیر resources/views قرار میدهیم. برای مثال یک فولدر در این مسیر به اسم layouts میسازیم و فایلی به اسم main.blade.php در آن قرار میدهیم:

<!DOCTYPE html>
<html lang="{{ config('app.locale') }}">
    <head>
        <meta charset="utf-8">
        <meta http-equiv="X-UA-Compatible" content="IE=edge">
        <meta name="viewport" content="width=device-width, initial-scale=1">

        <title>Main layout - @yield('title')</title>

        @yield('css')

    </head>
    <body>
        <h1>My layout</h1>
        <div id="nav">
            <ul>
                <li><a href="{{ url('/main/welcome') }}">welcome</a></li>
                <li><a href="{{ url('/main/first') }}">first view</a></li>
                <li><a href="{{ url('/main/secound') }}">secound view</a></li>
            </ul>
        </div>

        @yield('content')

        @yield('js')
    </body>
</html>

 

حالا سه فایل view با نامهای زیر میسازیم و در فولدری به نام main در مسیر resurces/views قرار میدهیم:

1- welcome.blade.php:

@extends('layouts.main')

@section('title', 'Welcome')

@section('css')
<!-- Styles -->
<style>
    html, body {
        background-color: #fff;
        color: orange;
    }
</style>
@endsection



@section('content')
<h1>Welcome</h1>
@endsection



@section('js')
<script>
    alert('Welcome');
</script>
@endsection

2- First.blade.php:

@extends('layouts.main')

@section('title', 'First')

@section('css')
<!-- Styles -->
<style>
    html, body {
        background-color: #ff0;
        color: #fff;
    }
</style>
@endsection



@section('content')
<h1>First</h1>
@endsection



@section('js')
<script>
    alert('First');
</script>
@endsection

3- Secound.blade.php:

@extends('layouts.main')

@section('title', 'Secound')

@section('css')
<!-- Styles -->
<style>
    html, body {
        background-color: #0ff;
        color: #f0f;
    }
</style>
@endsection



@section('content')
<h1>Secound</h1>
@endsection



@section('js')
<script>
    alert('Secound');
</script>
@endsection

 

حالا route هایی که برای نمایش view های بالا مینویسیم به این صورت خواهد بود:

Route::get('/main/welcome', function(){
	return view('main.welcome');
});

Route::get('/main/first', function(){
	return view('main.first');
});

Route::get('/main/secound', function(){
	return view('main.secound');
});

 

میتوانیم route های بالا را در یک Route::group قرار دهیدم:

Route::group(
    [
        'namespace' => 'main',
        'prefix' => 'main',
        'middleware' => ['auth'],
    ],
    function(){
        Route::get('/welcome', function(){
            return view('main.welcome');
        });

        Route::get('/first', function(){
            return view('main.first');
        });

        Route::get('/secound', function(){
            return view('main.secound');
        });
    }
);

 

میتوان برای هر route یک نام تعیین کرد:

Route::group(
    [
        'namespace' => 'main',
        'prefix' => 'main',
        'middleware' => ['auth'],
    ],
    function(){
        Route::get('/welcome', 'HomeController@welcome')->name('welcome');
        Route::get('/first', 'HomeController@first')->name('first');
        Route::get('/secound/{id}', 'HomeController@secound')->name('secound');
    }
);

بعد در view ها با استفاده از نام route، بصورت زیر به لینک دهی رو انجام میدیم:

<a href="{{ route('welcome') }}">welcome</a>
<a href="{{ route('first') }}">first</a>
<a href="{{ route('secound', ['id' => $article->id]) }}">secound</a>
